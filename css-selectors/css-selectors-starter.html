Cascading Style Sheets

Syntax

Each property/value pair is considered a CSS Declaration. Multiple declarations are separated by semi-colons. The selector defines which elements are affected by the rule.

For more information about CSS Syntax see w3schools.com. or the W3C Specfication

CSS Rules

CSS rules are statements that define the style of an element or group of elements.

Style rules consist of two parts:

  • Property
  • Value

CSS Selectors

The selector defines which elements are affected by the rule.

Types of Selectors

  • Type
  • Descendent
  • Child
  • Class
  • ID
  • Attribute
  • Universal
For more information about class and id's see w3schools.com.
