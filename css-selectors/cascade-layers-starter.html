Understanding CSS Layers

Cascade layers allow developers to structure and organize their CSS and control how style rules cascade. By creating "layers," developers can set parameters or scope around what rules cascade. Layers can be re-ordered without worrying about altering the default cascade rules, thereby elminating unwanted inheritance.

Rules within a single cascade layer cascade together, without interleaving with style rules outside the layer.

Common Use cases for cascade layers

  • Overwriting frameworks
  • Utility classes that override style guide rules
