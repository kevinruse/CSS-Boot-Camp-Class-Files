Flow Layout

  1. Run this file in the browser.
  2. Note the image is inside of a div, and the div has a 1-pixel border.
  3. Notice the gap of space between the border and the bottom of the image

    4. This is because the image element is rendered in the flow layout mode which means the browser is leaving space around it like it was a word in a sentence, flowing like an eBook or other flexible document type would flow.

  4. Change the layout mode of the image by removing the comments around the image CSS style rule.
  5. Refresh the file in the browser and the space is removed.

    6. The display: block value sets the layout mode of the element.
