CSS Writing Modes

مرحبًا بك في الخصائص المنطقية ، حيث يتم استبدال الأبعاد المادية العادية بأبعاد منطقية مشتقة من وضع كتابة اللغة مثل من اليسار إلى اليمين أو من اليمين إلى اليسار.

يمكننا استخدام خاصية وضع الكتابة للتأكد من عرض هذا النص بطريقة قابلة للقراءة ومناسبة للغة المكتوبة بها.

Welcome to logical properties, where regular physical dimensions are replaced by logical dimensions derived from language writing mode such as left-to-right or right-to-left.

We can use the write mode feature to ensure that this text is displayed in a readable and appropriate way for the language it is written in.
