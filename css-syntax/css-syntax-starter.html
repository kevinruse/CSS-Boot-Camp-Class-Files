The Cascade

Web designers can define style rules in three different places.

Where are CSS styles defined?

  1. Inline in an element
  2. In an embedded style sheet
  3. In an external style sheet

Embedded Style Sheets

These style sheets are written in the head section of an HTML page.

External Style Sheets

These style sheets are created in separate documents with a .css extension and it is simply a listing of css rules. It does not contain any HTML

Inline Styles

Inline styles are created by adding a style attribute to an html tag. The value of the style element is one or more property/value pairs separated by semi-colons.
